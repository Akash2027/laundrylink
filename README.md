# LaundryLink - Smart Laundry Marketplace

A full-stack laundry marketplace connecting customers with local laundry providers.

## Features

- Customer and Provider roles
- Real-time order tracking
- GPS-based provider discovery
- Image evidence upload
- Dispute resolution system
- Premium subscription with discounts

## Tech Stack

- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL (Neon.tech)
- **Storage:** Cloudinary
- **Deployment:** Vercel (Frontend), Render (Backend)
